== Description ==
Fastest way to create Gallery from your images.Ideal if you have large number of images and you change images of Galleries fequently. Easy maintenance and Gallery image change will no impact on WordPress database and files. Each Gallery image has ALT text for better Image SEO.
<strong>Steps to create Gallery</strong>
<ul><li>Create a folder inside your WordPress installation where you will keep all your Gallery images</li>
<li>Create Sub folders for each Gallery and add images you want in that Gallery.</li>
<li>Enter this Sub Folder Name on the New Visual Editor button option added by this plugin</li></ul>
<strong>Features</strong>
Plugin supports two Gallery modes Film Strip and Slide Show.

== Installation ==

The plugin can be installed like any other plugin.You can search from WordPress Dashboard (Udinra Image Sitemap) and click install to install it.You can also download it and upload it to your server wp-content/plugins directory and extract the zip file.
